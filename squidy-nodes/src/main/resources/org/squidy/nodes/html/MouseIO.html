Short Description

The Mouse I/O node is an Input and Output node. It receives as Input Node mouse events such as a mouse movements and mouse clicks on the Input Window (starts automatically). These mouse events are published as DataPosition2D and also as DataButton on the Output Pin. The Mouse I/O node can also be used as an Output node since it emulates mouse movements and button presses on the OS layer when it receives DataPosition2D or DataButton on the Input Pin. An example for such a mouse emulation and the corresponding DataButton objects can be seen here...

Code Sample for a left mouse click on Position P(x/y):

DataPosition2D dataPosition2D = new DataPosition2D(className.class, "mouse", x, y);
publish(dataPosition2D);
DataButton leftButtonPressed = new DataButton(className.class, "mouse left button click", DataButton.BUTTON_1, true));
publish(leftButtonPressed);
DataButton leftButtonReleased = new DataButton(className.class, "mouse left button click", DataButton.BUTTON_1, false));
publish(leftButtonReleased);

Sometimes it is usefull to give Java a small time between consecutive events to process them correctly. This can be done with a Thread.sleep(t)...
// short break for processing the mouse event
try {
    Thread.sleep(50);
} catch (InterruptedException e) {
    e.printStackTrace();
}
