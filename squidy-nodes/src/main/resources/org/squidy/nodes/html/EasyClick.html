Short Description

The EasyClick node helps to minimize the effects of hand movement while clicking with a absolute pointing device in mid-air e.g. a laserpointer. Normally when clicking with a laserpointer, the laserpointer is rotated a little bit throug hand pressure. Thus, the user clicks below the desired position. EasyClick compensates this rotation.

Node Type: Filter

Receives on Input Pin: DataPosition2D, DataButton

Publish on Output Pin: (modified) DataPosition2D, DataButton
