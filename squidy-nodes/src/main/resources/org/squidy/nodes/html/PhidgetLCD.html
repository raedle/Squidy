Short Description

The PhidgetLCD node is a driver for the LCD Panel on the Phidget microcontroller board http://www.phidgets.com/. The node sets the LCD text with the recently received value of a DataString object.

Code Sample for a DataString object that sets the LCD when received on the Input Pin:
DataString text = new DataString(className.class, "LCD-Text", "Hello World!");
publish(text);

Node Type: Output

Receives on Input Pin: DataString (the text to be print on the LCD)

Publish on Output Pin: none specific (only looping of the incoming data, if connected)
