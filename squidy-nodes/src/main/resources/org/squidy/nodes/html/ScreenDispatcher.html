Short Description

The ScreenDispatcher node is a bridge between Squidy and the iPhoneApp or similar applications that accepts an image stream on a tcp port. The node captures the local screen and transmit it to a remote computer or cell phone.

Node Type: Output

Receives on Input Pin: DataPosition

Publish on Output Pin: none specific (only looping of the incoming data, if connected)
