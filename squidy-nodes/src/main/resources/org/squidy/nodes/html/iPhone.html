Short Description

The iPhone node is a bridge between the Apple iPhone / iPod touch and Squidy. If the iPhone App "Squidy Client" is started, it sends touch positions, touch gestures and inertia data of the accelerometer via the OSC network protocol to this iPhone node in Squidy. Alternatively, also the more generic OSC node can be used to receive the interaction data from the iPhone App.

Node Type: Input

Receives on Input Pin: none

Publish on Output Pin: DataPosition2D, DataButton, DataInertial
