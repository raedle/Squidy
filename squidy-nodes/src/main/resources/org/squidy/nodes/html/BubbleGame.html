Short Description

The BubbleGame node is a bridge between Squidy and the BubbleGame, a dedicated application for performance evaluation for input devices based on Fitts' Law and Tapping Tasks specified in ISO 9241-9.

Node Type: Output

Receives on Input Pin: DataPosition2D (e.g. mouse position or finger contact), DataButton (selection status, e.g. mouse button)

Publish on Output Pin: none specific (only looping of the incoming data, if connected)
