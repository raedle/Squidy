Short Description

The SquidyRemote node streams any interaction data over the network based on the OSC protocol. Thus, it is a bridge between two Squidy applications or between Squidy and any hardware device or software that uses the OSC network protocol to transmit interaction data. The SquidyRemote node can be used both as sender or receiver of OSC messages depending on the connected node pin (input, output or both) and the status of the parameters "receiver" and "sender".

Node Type: Input and Output

Receives on Input Pin: any data type

Publish on Output Pin: any data type
