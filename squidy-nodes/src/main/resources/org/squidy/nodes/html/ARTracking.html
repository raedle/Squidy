Short Description

The ARTracking node receives interaction data from the ART Tracking System and publish them on the Output pin.

Node Type: Input

Receives on Input Pin: none

Publish on Output Pin: DataPosition3D (e.g. a single marker), DataPosition6D (e.g. rigid tracking body), DataButton (e.g. from flystick)
