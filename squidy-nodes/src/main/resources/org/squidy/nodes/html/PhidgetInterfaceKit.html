Short Description

The PhidgetInterfaceKit node is a driver for the Phidget microcontroller boards http://www.phidgets.com/. The node reads out sensor values of the connected sensors and publish them as DataAnalog objects on the Output Pin. On the Input Pin the status of connected LEDs or actuators can be set with DataDigital objects and specific attribute settings.

Code Sample for setting led on sensor pin 0 on microcontroller on true:
DataDigital digital = new DataDigital(className.class, "led pin", true);
digital.setAttribute(DataConstant.SENSOR_OUTPUT_INDEX, 0);
publish(digital);

Node Type: Input and Output

Receives on Input Pin: DataDigital (to set a specififc pin on the microcontroller e.g. to control LEDs)

Publish on Output Pin: DataAnalog (values of a connected sensor)
