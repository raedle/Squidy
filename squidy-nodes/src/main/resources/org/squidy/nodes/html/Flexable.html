Short Description

The Flexable node works similar to the TUIO node. It is a bridge between Squidy and the Flexable Framework. However, Flexable recognizes when a finger was lifted and sends according empty alive messages.

Node Type: Output

Receives on Input Pin: DataPosition2D, DataToken

Publish on Output Pin: none specific (only looping of the incoming data, if connected)
