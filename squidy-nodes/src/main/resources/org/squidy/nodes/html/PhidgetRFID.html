Short Description

The PhidgetRFID node is a driver for the RFID sensor of the Phidget microcontroller board http://www.phidgets.com/. The node publish identified and lost RFID tags as DataDigital on the Output Pin.

Node Type: Output

Receives on Input Pin: none

Publish on Output Pin: DataDigital (Attribute TAG_VALUE contains the RFID identification, flag=true when RFID tag is detected, flag=false when RFID tag is lost)
