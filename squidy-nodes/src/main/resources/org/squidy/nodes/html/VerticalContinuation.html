Short Description

The VerticalContinuation node is a filter that is usefull for optical tracking systems where the border of the display is also the border of the sensing system. The filter sets the cursor to the border if a movement path goes outside the border and it was not tracked completely.

Node Type: Filter

Receives on Input Pin: DataPosition2D (incoming positions)

Publish on Output Pin: DataPosition2D (adjusted positions)
