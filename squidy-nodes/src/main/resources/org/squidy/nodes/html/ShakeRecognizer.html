Short Description

The ShakeRecognizer node recognize shake gestures in a data stream of DataInertial from accelerometer sensors such as the iPhone or the laserpointer. When a high acceleration is detected a DataString is published on the Outout Pin. The parameter "Shake Threshold" defines the needed intensity value for detecting a shake gesture (if it is too small, alsomeasurement noise is interpreted as shake gesture). The parameter "Shake Interval" defines the minimum time span between two detected shake events - it prevents multiple detections in one shake gesture. If the data stream is visualized in the PipeVisualization, the gray points show the vector length of the acceleration which is constant if the sensor is not moving.

Node Type: Filter

Receives on Input Pin: DataInertial (getting the sensor data of the accelerometer)

Publish on Output Pin: DataString (publishing a DataString when a shake gesture is recognized)
